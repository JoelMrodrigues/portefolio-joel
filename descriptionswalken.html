Walken, comment ça marche?

Pour commencer Walken qu'est ce que c'est?

Walken est un "Move to earn" ce qui veut dire en français "déplacer pour gagner". La le termes gagner parle de gagner de la cryptomonnaies .

Comment-ça marche?

Alors walken est un jeu sur mobile/tablette, dans ce jeu il y a plusieurs type de monnaie:
-Les Gemmes : les gemmes se gagnent en marchant, elles servent à débloquer l'accès pour passer de niveaux(mais ne passe pas les niveaux).
-Le walken : le walken est la monnaies du jeux/en dehors du jeu, on les gagnes en jouant, le walken sert à augmenter le niveaux de son personnage, acheter des personnages/des cosmetiques/des baies pour améliorer ces personnages ou tout simplement les déplacer hors du jeu et les revendre.
Le Solana : Le Solana la cryptomonnaies qui sert de "base" au projet. Le Solana est une cryptomonnaies qui peut être mise dans le jeu afin de d'acquérir des personnages ou des cosmétiques. On ne peut pas en gagner sur Walken.

Comment jouer?

Alors pour jouer c'est très simple vous allez dans la rubriques compétitions, vous sélectionnez la ligue dans laquelle vous voulez jouez (dépend du niveaux de vos/votre personnage) ensuite vous sélectionnez le personnage souhaité.
Ensuite vous avez trois choix de type de "course" il y a la strengh / la stamina ou speed vous choisissez en fonction de vos stats exemple:
si vous avez 30 de speed et 15 dans les deux autres vous choisissez la speed car c'est la ou votre personnage est le meilleur (le type de combat détermine quel type de stats compte plus que les autres).
Une recherche d'adversaire se fait puis le combat commence une bataille de stats est alors engagé comme dans l'exemple .
(il y a aussi un case nombre de pas maintenant)
